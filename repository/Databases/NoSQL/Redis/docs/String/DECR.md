# DECR

---

## 官方参考

[DECR官方文档参考](https://redis.io/commands/DECR/)

## 命令语法

> DECR key 

## 作用

对指定键的值进行-1操作

如果指定键的值不存在则会先新建该键, 设置值为0, 之后再进行-1操作

如果指定键的值存在, 且其值不是字符串类型, 或其值是字符串类型但无法表示为一个*整数*, 则会抛出异常

!> 该操作对数值的范围有限制, 限制为64位带符号整数(-2^63, 2^63-1)

## 复杂性

- 时间复杂度

  - O(1)

## 参数

- key

需要对值进行-1操作的键

## 返回结果

- Integer reply

执行-1操作后的值

## 示例

```bash
DECR key
```
