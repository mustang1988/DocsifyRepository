# String

---

?> 字符串是Redis中最基础的基本数据类型, 在Redis中字符串是二进制安全的, 这意味着Redis中的字符串可以包含任意类型的数据, 比如一张JPEG图片或是一个可序列化的Ruby对象. 单个字符串值允许的最大长度限制为512MB.
