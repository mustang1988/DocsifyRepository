# mean

---

## 简介

计算数组中数值的平均值

## 是否影响原引用内容

- 否

## 参数说明

```javascript
_.mean(array)
```

- array

  需要计算平均值的数组

## 所属分类

- [Math相关函数](/repository/Libraries/Lodash/Math.md#math相关函数)

## 示例

```javascript
const _ = require('lodash');
_.mean([4, 2, 8, 6]); // 5
```
