# reverse

---

## 简介

将数组反序

## 是否影响原引用内容

- 是

## 所属分类

- [Array](/repository/libraries/Lodash/Array.md#array相关函数)

## 示例

```javascript
const _ = require('lodash');
const data = [1, 2, 3];
console.log(_.reverse(data)); // [ 3, 2, 1 ]
console.log(data); // [ 3, 2, 1 ]
```
